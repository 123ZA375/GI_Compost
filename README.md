# GI_Compost: Dataset for Compost Germination Index Prediction
This repository stores two sets of sorted experimental datasets for aerobic compost maturity research, which are applied to GBDT-based seed germination index (GI) prediction in the submitted manuscript.

## Dataset Overview
Two separated experimental datasets are included:
1. **Physicochemical factor dataset**: Time-series data of temperature, moisture content, pH, EC, TN, TOC and corresponding GI values, used to analyze the correlation between compost physicochemical characteristics and germination index.
2. **Different raw material dataset**: Composting experimental data with various raw substrate materials to investigate material effects on compost GI.

## Data Source Statement
All original experimental data are obtained from published open-access academic papers, and detailed reference information has been cited in the submitted journal article.
